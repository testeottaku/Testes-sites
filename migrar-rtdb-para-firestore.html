Migrar Realtime Database para Firestore

Essa página lê os dados atuais do Realtime Database e grava no Firestore seguindo o novo mapeamento do projeto.

Use uma única vez e só depois publique os arquivos já convertidos para Firestore.

  • Coleções migradas: Usuarios, Pedidos, Atividades, Suporte, Atualizacoes, Produtos, Vendedores e admins.
  • Documentos únicos: Manutencao e Versao.
  • Notificações: RTDB Notificacoes/{uid}/{notifId} → Firestore Notificacoes/{uid}/items/{notifId}.
